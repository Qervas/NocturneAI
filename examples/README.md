# NocturneAI Examples

This directory contains examples demonstrating the capabilities of the NocturneAI agent framework.

## MCP Examples

The `mcp_example` directory contains examples that showcase the Model Context Protocol (MCP) capabilities:

- `mcp_basic_demo.py` - Basic demonstration of MCP functionality
- `edit_file_demo.py` - Demonstrates file editing capabilities
- `git_mcp_demo.py` - Shows Git integration through MCP
- `websearch_demo.py` - Demonstrates web search capabilities

## Running Examples

You can run these examples directly:

```bash
# Run a specific example
python examples/mcp_example/mcp_basic_demo.py

# Run the web search demo
python examples/mcp_example/websearch_demo.py
```
